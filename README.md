# swayamjagtap.github.io
A public learning journal documenting my journey with weekly blogs, projects as I progress

This repository contains my personal website hosted with GitHub Pages. It is my public learning log, portfolio, and blog where I document what I study, build, and improve each week.

## About

I am a TYDS student currently learning Python, data science, and related technical skills. This website is where I track my weekly progress, share small projects, and gradually build a strong portfolio in public.

## What this site includes

- Weekly blog posts about what I learned
- Mini projects and coding practice
- Reflections on mistakes, struggles, and improvements
- My growth from beginner Python to data science

## Why I made this

I wanted one place to document my journey consistently instead of learning silently. This site helps me stay accountable, practice writing, and create visible proof of progress over time.

Each weekly blog post will be added inside the `posts/` folder, and linked from `blog.html`.

## Goal

My goal is to turn this site into a strong student portfolio that shows consistency, practical learning, and real project work.
